# 🎮 ブロック崩しゲーム

Web Audio APIを使用したクラブミュージック付きのブロック崩しゲームです。

## ✨ 特徴

- 🎹 **クラブミュージック**: Web Audio APIで生成されたリアルタイムのクラブミュージック（128 BPM）
- 🎵 **効果音**: ブロック破壊、パドル反射、ゲームイベントごとの効果音
- 🐱 **猫のアニメーション**: 5個破壊ごとに喜ぶ猫、ミス時に悲しい猫が登場
- 🖱️ **ドラッグ可能な猫**: マウスで猫をドラッグして移動できます
- 🎨 **カラフルなデザイン**: グラデーション背景とカラフルなブロック
- 📊 **スコアシステム**: スコア、ライフ、レベルの管理

## 🎯 遊び方

- **← → キー**: パドルを左右に移動
- **スペースキー**: ゲーム開始 / 一時停止
- **F5キー**: ゲームオーバー時にリスタート
- **マウス**: 猫をドラッグして移動

## 🛠️ 技術スタック

- HTML5 Canvas
- JavaScript (ES6+)
- Web Audio API

## 📝 ゲームルール

1. パドルを操作してボールを跳ね返す
2. すべてのブロックを壊すとレベルクリア
3. ボールを落とすとライフが減少
4. ライフが0になるとゲームオーバー
5. ブロックの位置によってポイントが異なります

## 🎵 音楽について

ゲーム開始時にWeb Audio APIを使用したクラブミュージックが自動的に再生されます：
- キックドラム（4/4拍子）
- ハイハット
- ベースライン
- シンセパッド
